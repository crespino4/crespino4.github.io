Favorites Interaction Widget

Status

This is an example of how to display a list of common "favorite" transfer targets.

The list of corporate favorites is controlled using External Organizations and Contacts.

Create an External Organization named "Favorites". Then within that organization create as many External Contacts as you'd like to represent the items that should be listed as "Favorites" to the agent.

For a Queue favorite enter "Queue" as the first name and the name of the queue as the last name
