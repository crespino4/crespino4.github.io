Overview

Premium applications are AppFoundry partner applications running inside Genesys Cloud. Premium app is integrated with Genesys Cloud's billing system. The wizard is a simple web app built to provision additional items in Genesys Cloud for use by the Premium App. The Premium App Starting Guide explores the sample Premium App Wizard installer and how partners can leverage it for their own Premium Apps.
